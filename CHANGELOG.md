# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.1.0] - 2026-01-05

### Added
- **Folder copy support** - Copy entire shared folders with all contents
- Recursive folder copying (copies all subfolders)
- Visual distinction for folders (📁 icon, blue highlight)
- Folder item count display

### Changed
- File list now shows 📄 for files and 📁 for folders
- "X items" display for folders instead of file size

## [1.0.0] - 2026-01-01

### Added
- Initial release by [queery-id](https://github.com/queery-id)
- Copy files from "Shared with me" to My Drive
- Target folder selection (My Drive root + subfolders)
- File type filtering (Documents, Spreadsheets, PDFs, Images)
- **Sort by options:**
  - Date (Newest/Oldest First)
  - Name (A-Z / Z-A)
  - Size (Largest/Smallest First)
  - Owner (A-Z)
- Skip duplicates option
- Multi-select with checkboxes
- Real-time progress bar
- Results summary (success/skipped/failed)
- Modern responsive UI with gradient design
- Mobile-friendly layout
- Support for up to 500 shared files
